# 컴포넌트 인벤토리 (web/mobile 패리티)

> 공유 컴포넌트의 계약과 플랫폼별 구현 상태를 추적한다.
> 상태: ✅ 완료 · 🚧 진행 · ⬜ 미착수 · — 해당 없음

| 컴포넌트 | 계약(요지) | web | mobile | 비고 |
|----------|-----------|-----|--------|------|
| Button | variant/size/disabled | ⬜ | ⬜ | |
| Card | 자산/시뮬 카드 컨테이너 | ⬜ | ⬜ | |
| Money | 금액 표기(통화·부호·tabular) | ⬜ | ⬜ | 수익/손실 색 토큰 |
| Chart | 라인/영역/캔들 (시세·백테스트) | ⬜ | ⬜ | web d3 · mobile Victory/ECharts |
| Table | 현금흐름·비교표 | ⬜ | ⬜ | |
